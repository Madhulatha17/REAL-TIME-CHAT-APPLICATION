# REAL-TIME-CHAT-APPLICATION

## PROJECT OVERVIEW

This project is a Real-Time Chat Application developed using React.js for the frontend and WebSockets for real-time communication. Users can send and receive messages instantly, and chat history is displayed within the interface.

## FEATURES

* Real-Time Messaging
* Responsive User Interface
* WebSocket Communication
* Message History
* Modern Chat Layout

## TECHNOLOGIES USED

* React.js
* Node.js
* Socket.IO
* CSS3

## HOW TO RUN

1. Install dependencies in both client and server folders.
2. Start the server.
3. Start the React application.
4. Open the browser and begin chatting.

## CONCLUSION

The application demonstrates real-time communication using WebSockets and provides a responsive chat experience with persistent message history during the session.
